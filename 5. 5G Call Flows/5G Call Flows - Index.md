# 5. 📲 5G Call Flows

This module brings everything together by analyzing the end-to-end signaling call flows between the UE, NG-RAN, and 5GC.

## 📋 Core Topics
- **Initial Registration & Attach:** UE authentication, security context establishment, and AMF registration.
- **PDU Session Establishment:** Protocol Data Unit (PDU) sessions, IP address allocation, and UPF path establishment.
- **Handovers:** Xn-based and N2-based handovers for mobility management.
- **Service Request & Triggering:** User plane connection reactivation from RRC_INACTIVE/RRC_IDLE states.
