# 1. 🌐 The 5G System

This module covers the fundamentals of the 5G system architecture, standardization bodies (like 3GPP), and the core drivers behind 5G technology.

## 📋 Core Topics
- **Introduction to 5G:** Evolution from 1G to 5G, key capabilities, and use cases (eMBB, URLLC, mMTC).
- **Standards & Specifications:** 3GPP releases (Release 15, 16, 17, and beyond).
- **5G System Architecture:** Overview of the user equipment (UE), Radio Access Network (RAN), and Core Network (5GC).

## 📝 Related Notes
- [[01. 5G System Standardization and Global Requirements]] — Global oversight (ITU), defining IMT-2020, and technology development (3GPP).
- [[02. 5G Use Cases and Performance Dimensions]] — Deep dive into the three pillars (eMBB, URLLC, mMTC) based on ITU-R M.2083 and detailed technical targets.
- [[03. 5G System Architecture and Core Functions]] — Exhaustive breakdown of User Equipment, 5G RAN base stations, 5G Core Network microservices (AMF, SMF, UPF, PCF, UDM, AUSF), and reference point interfaces.
- [[04. 5G System Deployment Options]] — In-depth mapping of 3GPP Standalone (SA) vs Non-Standalone (NSA) pathways, MR-DC dual connectivity, and Options 1 through 7 (with details on 3/3a/3x and 4/4a sub-options).
